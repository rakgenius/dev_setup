# Minikube and kubectl commands

## version
```bash
minikube version
```

## status
```bash
minikube status
```

## start minikube
```bash
minikube start
```

## stop minikube
```
minikube stop
```

## Delete minikube
```bash
minikube delete
```

## get context
```bash
ubectl config current-context
```

## get nodes
```bash
kubectl get nodes
```

## list pods
```bash
kubectl get pods --all-namespaces
```
